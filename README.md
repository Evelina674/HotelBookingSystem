# Hotel Booking System

## Опис

Консольний застосунок для управління бронюванням номерів готелю.

## Функціональність

- Управління клієнтами
- Управління номерами
- Бронювання номерів
- Формування рахунку
- Серіалізація JSON
- LINQ-запити
- Юніт-тестування

## Використані принципи ООП

- Інкапсуляція
- Наслідування
- Поліморфізм
- Абстракція

## Використані патерни

- Builder
- Decorator

## Технології

- C#
- .NET 9
- xUnit
- LINQ
- System.Text.Json

## Запуск

```bash
dotnet run --project HotelBookingSystem.App
```

## Запуск тестів

```bash
dotnet test
```
